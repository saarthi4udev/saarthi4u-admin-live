# Saarthi4u - Admin

Official repository for the Admin Panel of Saarthi4u.

## Prerequisites

- Node.js (v22.11.0 stable)  - [Download Link](https://nodejs.org/en/blog/release/v22.11.0)

## Steps to Run the Project

In the project directory, you can run:

- ### `npm i --force` or `npm install --force` - only first time, make sure to use --force flag to resolve dependency conflicts

- ### `npm run dev` - to run admin panel in development mode
